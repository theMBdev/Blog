### What is this

Its a blogging platform that users can sign up to use. It has a sign in/ sign up authentication and CMS were the user can create, edit, update and delete posts as well as update other information about their blog.

### Why was it built

After using a little php in a past project I wanted to learn more and decided to build a blog which would allow me to use crud functions along with a sql database. 

### Main technologies used
Javascript, Php, Ajax, SQL
